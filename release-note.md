# 2025-0４-２４ Release

## 機能

- ウィジェット新規作成時にご利用中のプランに応じて最適な地域別出し分けの設定を行うようになりました

## 機能

  - 再同意を求めたい場合、外部サービスに変更があった場合などにご活用ください
