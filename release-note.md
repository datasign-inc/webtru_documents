# 2023-06-28 Release

## 機能

- 二要素認証を設定できるようになりました
  - アカウント画面から有効にできます

## その他

- ウィジェット設定変更画面にモードに応じて必要な項目のみ表示するようにしました
- ウィジェットによるローカルストレージ使用量を削減しました
- ウィジェット設置タグのid属性を削除しました
  - id属性を含むタグも引き続き動作します
- いくつかの機能改善、バグ修正が適用されました

# 2023-06-14 Release

## 機能

- 公表モード埋込方式をカテゴリレベルで展開した状態で設置できるようになりました([マニュアル](https://webtru.zendesk.com/hc/ja/articles/15381571060249))
- ウィジェット画面にキーワード検索とソート機能を追加しました
- 5回連続してログインに失敗するとアカウントがロックされるようになりました
  - 再度ログインするには時間をおいてください

## その他

- 公表モード埋め込み方式を同一サイトに設置できる数の制約を改善しました
- サービス名が長い場合のウィジェット内の表示を改善しました
- ウィジェットのモードを作成画面、一覧画面に表記するようにしました
- 検査対象URLを巡回URLに変更しました
- いくつかの機能改善、バグ修正が適用されました
